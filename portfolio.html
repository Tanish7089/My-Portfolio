Zoom-Clone

This project is a video conferencing web app similar to Zoom, built using the MERN stack (MongoDB, Express.js, React.js, Node.js).
Frontend (React.js): Creates the user interface for joining meetings, chatting, and controlling video/audio.
Backend (Node.js + Express.js): Handles meeting creation, user authentication, and communication between clients.
Database (MongoDB): Stores user accounts, meeting details, and chat history so data is saved even after the app restarts.
WebRTC: Enables real-time video and audio streaming directly between users.
Socket.IO: Allows instant communication for chat messages, participant updates, and meeting controls.
REST APIs: Used for actions like creating accounts, starting meetings, and retrieving stored data.
In short, React manages what users see, Node.js + Express.js handle requests and logic, MongoDB stores information, and WebRTC + Socket.IO make live calls and chats possible. This combination allows smooth real-time meetings over the web.
